# Ruby Chess ♟️

A command-line chess game written in Ruby. This project implements the core structure of a playable chess engine, including piece classes, board logic, turn handling, and a YAML-based save/load system.

## Features
- Full 8x8 chess board rendered in the terminal
- Object-oriented architecture (Piece, Board, Game classes)
- Individual classes for each chess piece
- Move validation framework
- Turn-based gameplay (White vs Black)
- Save and load game state using YAML
- Algebraic-style move input (e.g., `e2 e4`)
